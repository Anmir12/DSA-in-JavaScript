# Data Structures & Algorithms Using JavaScript

A comprehensive collection of data structures and algorithms implementations in JavaScript.

## Contents

- **Data Structures**
    - Arrays
    - Linked Lists
    - Stacks data structure 
    - Queues 
    - Trees
    - Graphs
    - Hash Tables
    - Heap

- **Algorithms**
    - Sorting (Bubble, Quick, Merge,etc..,)
    - Searching (Binary, Linear, etc.)
    - Graph Traversal (BFS, DFS)
    - Dynamic Programming
    - Greedy Algorithms
    - Recursion

## Getting Started

1. Clone or download this repository
2. Navigate to the desired algorithm or data structure folder
3. Open files with `.js` extension to view implementations
4. Run examples with Node.js

```bash
node filename.js
```

## Prerequisites

- Node.js installed on your system
- Basic JavaScript knowledge

## Usage

Each file contains:
- Implementation of the data structure/algorithm
- Time and space complexity analysis
- Example usage
